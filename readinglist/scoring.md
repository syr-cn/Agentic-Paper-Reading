# Scoring Calibration

Use this file to keep scoring consistent over time.

## Example rubric reminder (100)
- Relevance: 40
- Novelty: 20
- Rigor: 20
- Actionability: 15
- Clarity/Reproducibility: 5

## Calibration notes
- If uncertain between 2 tiers, bias upward only with strong empirical evidence.
- Penalize papers that hide critical implementation details.
