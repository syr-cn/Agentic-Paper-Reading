# SkillRL 阅读笔记

## 0) Metadata
- **Title:** Evolving Agents via Recursive Skill-Augmented Reinforcement Learning
- **Alias:** SkillRL
- **Venue / Status:** arXiv 2602.08234
- **Links:**
  - HTML: https://arxiv.org/html/2602.08234v1
  - PDF: https://arxiv.org/pdf/2602.08234
  - Code: https://github.com/aiming-lab/SkillRL
- **Read depth:** skim

## 1) TL;DR
- 用技能库替代原始轨迹记忆，降低噪声与 token 开销。
- 提出 SkillBank + 自适应检索 + 递归演化机制，让技能库与策略共同进化。
- 在 ALFWorld、WebShop 和 7 个搜索增强任务上报告 SOTA，较强基线提升 15.3%。

## 2) Problem & Motivation
- 现有记忆多是原始轨迹缓存，冗余、噪声高，难以复用。
- 目标是将经验蒸馏成可迁移技能，支持持续策略改进。

## 3) Method
- Experience-based distillation 构建层次化 SkillBank。
- 检索时同时考虑通用启发和任务特化技能。
- 训练中递归更新技能库，实现 skill-policy co-evolution。

## 4) Key Evidence
- 报告相对强基线 +15.3% 提升。
- 声称复杂任务下鲁棒性更好。

## 5) Notes
- 值得深读点：SkillBank 的表示与更新触发条件。
