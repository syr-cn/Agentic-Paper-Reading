## 🔬 Top 3 研究进展（方法/模型/benchmark）

**1. 🔥 OmniMem — 自主研究管线发现的终身多模态 Agent Memory 框架（4/1 arXiv:2604.01007）**
- 用 Autoresearch pipeline 自主做 ~50 次实验，从朴素 baseline（F1=0.117）进化到 SOTA（F1=0.598），提升 +411%。关键发现：Bug fix > 架构改动 > Prompt engineering > 超参调优，颠覆传统 AutoML 认知。两个 benchmark：LoCoMo + Mem-Gallery
- 📎 <https://arxiv.org/abs/2604.01007> | 代码：<https://github.com/aiming-lab/OmniMem>
- 💡 **与你相关：** 直接竞品方向！autoresearch + 多模态 memory，可作为 MemOCR / ReMemR1 下一阶段的对标

**2. HippoCamp — 个人电脑上下文 Agent Benchmark（4/1 arXiv:2604.01221）**
- 评测 agent 在 2000+ 真实文件（42.4GB，多模态）中做搜索、证据感知、多步推理。最强商业模型仅 48.3% 准确率，主要瓶颈在长程检索和跨模态推理
- 📎 <https://arxiv.org/abs/2604.01221> | 项目页：<https://hippocamp-ai.github.io/>
- 💡 **与你相关：** 新的 agent memory benchmark，比 LoCoMo 更贴近真实场景，值得作为评测集

**3. "Therefore I am. I Think" — 推理模型先决策后思考？（4/1 arXiv:2604.01202）**
- 用线性 probe 从激活值中解码工具调用决策，发现在生成任何推理 token 之前决策已编码。Activation steering 验证因果性：chain-of-thought 在多数情况下是决策后的"合理化"而非真正推理
- 📎 <https://arxiv.org/abs/2604.01202>
- 💡 **与你相关：** 对 reasoning model 的 memory-driven planning 机制有根本性启示，影响 RL reward 设计
