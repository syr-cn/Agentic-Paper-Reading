# MemAtlas: SYR's Memory Paper Reading

A curated repository for memory-centric AI paper reading, tracking, and synthesis.

## Focus Areas
- Agentic memory systems
- Long-context reasoning
- Retrieval-augmented reasoning
- Multi-modal agents
- Reinforcement learning for agents
- Scientific literature understanding

## Repository Structure
- `PAPER_PREFERENCES.md` — paper selection taste and ranking criteria
- `papers/` — raw paper records and daily/weekly captures
- `notes/` — structured reading notes (per paper)
- `readinglist/` — prioritized reading queues
- `weekly/` — weekly summaries and trend analysis

## Workflow
1. Add candidates into `papers/` or `readinglist/`
2. Read and write structured notes in `notes/`
3. Produce weekly synthesis in `weekly/`
4. Iterate preference signals in `PAPER_PREFERENCES.md`

## Naming Rationale
Following concise GitHub naming styles (*Atlas/*Bench/*Agent), **MemAtlas** represents a map of the memory-research landscape.
