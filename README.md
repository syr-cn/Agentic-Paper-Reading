# MemAtlas: SYR's Memory Paper Reading

A curated reading repository for memory-centric AI research, including:
- agentic memory systems
- long-context reasoning
- retrieval-augmented reasoning
- multimodal memory for intelligent agents

## Scope
- Daily/weekly paper digests
- Structured reading notes
- Key idea comparisons
- Open problems and future directions

## Naming rationale
Following common GitHub naming taste (short + brandable, like *Atlas/*Bench/*Agent),
**MemAtlas** represents a map of the memory-research landscape.
